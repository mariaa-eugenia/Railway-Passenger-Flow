# UK Rail Passenger Trends: Recovery & Forecast

This project analyzes UK railway station-level passenger flow from **2022 to 2024**, using official data from the **Office of Rail and Road (ORR)**. It explores historical trends, highlights the busiest stations, and forecasts 2024–25 demand using machine learning.

![Dashboard Preview](./dashboard_preview.png)

---

## Dashboard Features

- **Top Stations by Passenger Volume**
- **Year-on-Year Growth Comparison**
- **Forecast for 2024–25 (Linear Regression)**
- **Map View of Passenger Totals by Location**
- **Clean & Interactive Power BI Visuals**

---

## Data Sources

- `Entries_Exits_Interchanges_2022-23.csv`
- `Entries_Exits_Interchanges_2023-24.csv`
- `Region mapping.xlsx`

Data was transformed using **Python (Pandas + Scikit-learn)** and visualized in **Power BI**.

---

## How to Reproduce

1. Clone the repo  
2. Run the Jupyter Notebook: `Railway Analysis Notebook.ipynb`
3. Open the `.pbix` file in Power BI Desktop  
4. Explore the dashboard!
